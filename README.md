# Alphabet Adventure

A playful two-step web activity that helps young children learn alphabet starting sounds.

## Features

- **Step-by-step learning:** Start with five simple multiple-choice questions about word beginnings, then move to a matching game with trickier letters.
- **Friendly visuals:** Emoji-based imagery and warm colors keep the focus on fun.
- **Positive feedback:** Spoken and written messages celebrate every correct answer.
- **Progress tracking:** See how many letters have been mastered at any time.
- **Replayable:** Restart and shuffle to practice again with a new order.

## Getting Started

1. Open `index.html` in your browser.
2. Follow the prompts to answer the starting-letter questions.
3. Match pictures with their starting letters.
4. Celebrate when the adventure is complete!

The experience is fully client-side and requires no additional setup.
